<Note>
The WebSocket TTS endpoint enables bidirectional streaming for low-latency text-to-speech generation with MessagePack serialization.
</Note>

<Warning>
For long-running generations (>20 seconds), increase WebSocket timeout values to prevent disconnection:
- `keepalive_ping_timeout_seconds` (default: 20)
- `keepalive_ping_interval_seconds` (default: 20)
</Warning>
