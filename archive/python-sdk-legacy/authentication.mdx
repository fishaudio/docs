## Session Initialization

Initialize a `Session` with your API key to start using the Fish Audio SDK:

```python
from fish_audio_sdk import Session

# Initialize with your API key
session = Session("your_api_key")
```

### Using Environment Variables

For better security, store your API key in environment variables:

<Tabs>
  <Tab title="Using os">
    ```python
    import os
    from fish_audio_sdk import Session

    session = Session(os.getenv("FISH_API_KEY"))
    ```

    Set the environment variable in your shell:
    ```bash
    export FISH_API_KEY=your_api_key_here
    ```
  </Tab>
  <Tab title="Using dotenv">
    ```python
    import os
    from dotenv import load_dotenv
    from fish_audio_sdk import Session

    # Load environment variables from .env file
    load_dotenv()

    session = Session(os.getenv("FISH_API_KEY"))
    ```

    Create a `.env` file in your project root:
    ```bash
    FISH_API_KEY=your_api_key_here
    ```

    <Tip>
      The `dotenv` module comes from the [python-dotenv](https://pypi.org/project/python-dotenv/) package
    </Tip>
  </Tab>
</Tabs>

### Custom Endpoints

If you need to use a proxy or custom endpoint:

```python
session = Session(
    apikey="your_api_key",
    base_url="https://your-proxy-domain.com"
)
```
