## Requirements

- Python 3.9 or higher

## Install

It is strongly recommended to use a virtual environment manager to avoid dependency conflicts and ensure reproducible installations.

<Tabs>
  <Tab title="uv">
    ```bash
    uv pip install fish-audio-sdk
    ```
  </Tab>
  <Tab title="poetry">
    ```bash
    poetry add fish-audio-sdk
    ```
  </Tab>
  <Tab title="conda">
    ```bash
    conda install -c conda-forge fish-audio-sdk
    ```
  </Tab>
  <Tab title="pip">
    ```bash
    pip install fish-audio-sdk
    ```
    <Warning>
      Using pip without a virtual environment may cause dependency conflicts. Consider using uv, poetry, or conda instead.
    </Warning>
  </Tab>
</Tabs>
